📝 Zapisywanie wyrażeń

Zamieniamy słowa na symbole matematyczne:

„o 6 większa od x" → x + 6
„o s mniejsza od n" → n − s
„9 razy mniejsza od y" → y : 9 lub y/9
„4 razy większa od (2−x)" → 4 · (2−x)
⚠️ PUŁAPKA: „4 razy większa od 2−x" = 4·(2−x), NIE 4·2−x! Nawiasy!

✂️ Upraszczanie (zbieranie wyrazów podobnych)

Możemy dodawać/odejmować tylko wyrazy z TĄ SAMĄ literą i potęgą:

2x + 4x = 6x ✓
2a − 5a = −3a ✓
3y + y − 2y = 2y ✓ (pamiętaj: y = 1y)
4z + 0,6z = 4,6z ✓
−3x + 2x + 5x = 4x ✓ (kolejność nie ma znaczenia!)
🔑 Wyrazów z różnymi literami (np. 3x i 2y) NIE łączymy!

🔢 Obliczanie wartości wyrażenia

Wstawiamy podane liczby zamiast liter i obliczamy krok po kroku:

3x + 4 dla x = −4:
= 3·(−4) + 4
= −12 + 4
= −8 ✓

−8 − 2y dla y = 5:
= −8 − 2·5
= −8 − 10
= −18 ✓

2x + 3y − 6 dla x=−3, y=2:
= 2·(−3) + 3·2 − 6
= −6 + 6 − 6
= −6 ✓

(1/3)·a·(a−2) dla a=9:
= (1/3)·9·(9−2)
= 3·7
= 21 ✓
⚠️ Ujemna liczba w nawiasie: 3·(−4) = −12, nie +12!

🚫 Najczęstsze błędy Varvarы z poprzedniej kartkówki

❌ (−1/3)·18 = 6 → ✓ = −6 (znak!)
❌ 2·(−3) + 3·2 − 6 = −6+6−6 = 0 → ✓ = −6
❌ (1/3)·9·7 = 28 → ✓ (1/3)·9 = 3, potem 3·7 = 21
❌ 4·(2−x) zapisane bez nawiasów
