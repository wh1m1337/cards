Wyrażenia algebraiczne

Klasa 6 · Przygotowanie do kartkówki

📌 Co to jest wyrażenie algebraiczne?

Wyrażenie algebraiczne to zapis matematyczny zawierający liczby, litery (zmienne) i działania (+, −, ·, :).

Przykłady wyrażeń

3a + 5 2x − 7 4(n + 2) a² + b

💡 Pamiętaj!

Litera w wyrażeniu to nieznana liczba – zastępuje dowolną wartość liczbową.

✏️ 1. Zapisywanie wyrażeń algebraicznych

Jak zamieniać słowa na zapis algebraiczny:

  • suma liczby a i 5 → a + 5
  • różnica x i 3 → x − 3
  • iloczyn 4 i n → 4n (zamiast 4·n)
  • iloraz b i 2 → b : 2 lub b/2
  • liczba zwiększona o 7 → n + 7
  • liczba zmniejszona o 7 → n − 7
  • liczba dwukrotnie większa → 2n
  • liczba trzy razy mniejsza → n : 3
Przykład zadania

„Cena jabłka wynosi a złotych. Kupuję 3 jabłka i płacę 2 zł więcej za torbę."

→ 3a + 2

🔢 2. Obliczanie wartości wyrażeń

Podstawiamy konkretną liczbę zamiast litery i obliczamy.

Krok 1: Podstaw liczbę zamiast litery

Krok 2: Oblicz zgodnie z kolejnością działań

Kolejność: nawiasy → potęgi → · i : → + i −

Przykład

Oblicz wartość wyrażenia 3a + 2 dla a = 4

→ 3 · 4 + 2 = 12 + 2 = 14

Przykład z dwoma zmiennymi

Oblicz 2x − y dla x = 5, y = 3

→ 2 · 5 − 3 = 10 − 3 = 7

✨ 3. Upraszczanie wyrażeń algebraicznych

Upraszczamy przez łączenie wyrazów podobnych (tych samych liter lub samych liczb).

  • PODOBNE 3a i 5a → można dodać → 8a
  • RÓŻNE 3a i 5b → nie można łączyć
  • LICZBY 4 i 7 → można dodać → 11
Przykład 1 – prosta suma

3a + 5a = 8a

→ (3 + 5)a = 8a

Przykład 2 – mieszane wyrazy

Uprość: 4x + 3 + 2x − 1

→ 4x + 2x + 3 − 1 = 6x + 2

Przykład 3 – różne litery

Uprość: 5a + 3b + 2a − b

→ (5a + 2a) + (3b − b) = 7a + 2b

⚠️ Uwaga!

Wyrazy z różnymi literami (a i b) lub literami z różnymi wykładnikami (a i a²) NIE są podobne i nie można ich łączyć!
